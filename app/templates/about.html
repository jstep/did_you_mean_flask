About this project

This app mimics the functionality of popular "Did you mean?" word suggestion features such as when a misspelled word is entered into a search engine.

It uses a Levenshtein Distance algorithm which calculates a cost of changing one string, letter by letter, to another. Also known as Edit Distance it recursively loops over two strings and assigns a cost to change one to the other. The operations are substitution, insertion, and deletion. Each is set to a cost of 1 per operation, but these could be adjusted to weight different operations, thus incurring a larger cost.

I initially wrote it as a command line program then built this Flask app around it to make it nicer to use.
